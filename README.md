# Neural Style Transfer using TensorFlow

This project implements Neural Style Transfer using TensorFlow and TensorFlow Hub.

Neural Style Transfer is a deep learning technique that combines the content of one image with the artistic style of another image.

For example, a photo can be transformed to look like a famous painting.

Technologies Used:
- Python
- TensorFlow
- TensorFlow Hub
- Matplotlib
- Google Colab

Project Workflow:
1. Upload content image
2. Upload style image
3. Apply pretrained Neural Style Transfer model
4. Generate stylized output image

Model Used:
Arbitrary Image Stylization v1 from TensorFlow Hub.

Output:
The generated image contains the original content with the artistic style applied.

Example Result:
(Content Image + Style Image → Stylized Image)
